# Note Taker
  ## Description
Note Taker is a front and back end application hosted on Heroku. It contains both data persistance using HTTP protocals to save data as well as validation to ensure minimal operating errors.

## Installation
Download source files from GitHub, install dependencies with `npm i` and then launch on localhost:1217.
## Usage
Write note titles and wrote notes beneath. Hit the save disk to add it to the list of notes, hit the plus icon to add a new note. You can delete notes using the trash can icon.
